## 📁 How to Use

To run this portfolio locally:
1. Clone the repo  
2. Open `index.html` in your browser  
3. No dependencies required — it's fully static

To deploy:
- Push to GitHub
- Enable GitHub Pages under Settings → Pages
- Choose `main` branch and root `/`
